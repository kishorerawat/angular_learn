This example shows all the filters available in Angular

Filter lowercase : {{(firstName | lowercase) + " "}}{{lastName | lowercase}}

Filter uppercase : {{(firstName | uppercase) + " "}}{{lastName | uppercase}}

Filter currency for amounts : {{10000 | currency:'₹'}} or {{20000 | currency:'₨'}}

Filter orderBy for sorting lists :

  • {{x.name + " , " + x.relation}}

Filter filter for showing subset of lists :

  • {{x.name + " , " + x.relation}}

Filter using ng-model in the expression :

City :

  • {{x.city + " , " + x.state}}
