AngularJS provides many built-in services like $location, $http etc.

This example shows the usage of these various services

Service : $location

location.absUrl : {{myAbsUrl}}

location.url : {{myUrl}}

location.protocol : {{myProtocol}}

location.host : {{myHost}}

location.hostWithPort : {{myHostWithPort}}

Service : $http

Today's Welcome Message is (retrieved by $http call) :

{{myWelcome}}

Service : $timeout

This header will change after two seconds :

{{myHeader}}

Service : $interval

This service will display time every second :

Time : {{theTime}}
